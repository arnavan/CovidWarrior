## Prerequisites

To run this app on your pc you would need  couple of python modules.

Install them with the command

```
pip install kivy
```
```
pip install kivymd
```
```
pip install pandas
```
```
pip install covid
```
```
pip install requests
```
```
pip install akivymd
```
